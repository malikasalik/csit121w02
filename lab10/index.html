What is XML? XML stands for eXtensible Markup Language, and is a coding language that operates on user-defined tags to create meaningful data which allows computers to identify relationships between different items in the document. It is the "tags" which makes XML so useful because XML is just information wrapped in tags.
How is XML used? It is used to structure and describe information. It provides a method for transferring textual content across mediums (browser, mobile, etc.). It is bascially platform independent and is designed and used to store and transfer data.

JavaScript
What is JavaScript? Javascript is a programming language that runs inside the browser and works together with HTML and CSS to create interactive experiences.
How is JavaScript related to HTML & CSS? JavaScript is closely related to HTML and CSS as it is often used together to create dynamic and interactive web pages. HTML provides the structure and content of a webpage, CSS is used for styling and layout, and JavaScript adds interactivity and functionality to the webpage. With JavaScript, you can manipulate HTML elements, respond to user actions, etc.
